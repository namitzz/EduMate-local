üöÄ Quick Deployment Options

üê≥ Option 1: Docker (Recommended)

Easiest setup - everything runs in containers:

$ git clone https://github.com/namitzz/EduMate-local.git
$ cd EduMate-local
$ docker compose up --build

Then open http://localhost:8501 in your browser.

üñ•Ô∏è Option 2: Local Installation

For manual setup without Docker:

# Install Ollama first from https://ollama.com
$ ollama pull mistral

# Terminal 1 - Backend
$ cd backend
$ pip install -r requirements.txt
$ python ingest.py
$ uvicorn main:app --reload --port 8000

# Terminal 2 - Frontend
$ cd ui
$ pip install -r requirements.txt
$ streamlit run app_public.py

‚òÅÔ∏è Option 3: Cloud Deployment

Deploy to DigitalOcean, AWS, or any VM with Docker:

  1. Create a VM with 4GB+ RAM and Ubuntu 22.04
  2. Install Docker and clone the repository
  3. Run docker compose up -d
  4. Configure reverse proxy for HTTPS (optional)

See DEPLOYMENT.md for detailed instructions.

üåê Option 4: Streamlit Cloud (Free)

Deploy your Streamlit app for free:

  1. Fork this repository
  2. Sign up at streamlit.io/cloud
  3. Deploy directly from your GitHub fork
  4. Configure environment variables

Note: You'll need to deploy the backend separately or use a cloud LLM API.
